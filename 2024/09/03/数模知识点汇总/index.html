层次分析法

层次分析法可以利用树状的层级结构，将复杂的决策问题在一个层级中区分为数个简单的子问题，并且每个子问题可以独立进行分析，这个层级中的子问题可以包含是任何类型的子问题，无论是有形的还是无形的，仔细计算的或者粗略估计的，理解清晰或模糊的，只要是用于最终决策的子问题都可以包括于此

一旦这个层级建立完毕，决策专家会有有系统地评估尺度针对每一个部分的相对重要性给予权重数值，其后建立成对比较矩阵，并求出特征向量及特征值，以该特征向量代表每一层级中各部分的优先权，能提供决策者充分的决策资讯并组织有关决策的评选条件或标准（criteria）、权重（weight）和分析（analysis），且能减少决策错误的风险性。

皮尔逊相关系数适用范围：

当两个变量的标准差都不为零时，相关系数才有定义，皮尔逊相关系数适用于：

(1)、两个变量之间是线性关系，都是连续数据。

(2)、两个变量的总体是正态分布，或接近正态的单峰分布。

(3)、两个变量的观测值是成对的，每对观测值之间相互独立。

斯皮尔曼相关系数适用范围：

斯皮尔曼等级相关系数对数据条件的要求没有皮尔逊相关系数严格，只要两个变量的观测值是成对的等级评定资料，或者是由连续变量观测资料转化得到的等级资料，不论两个变量的总体分布形态、样本容量的大小如何，都可以用斯皮尔曼等级相关系数来进行研究。

故一般先用QQ图，看是否大致符合正态分布，再使用检验方法进一步确定，若不符合正态分布，则使用斯皮尔曼相关系数，否则使用皮尔逊相关系数

Shapiro-Wilk 检验是用于检验一个样本是否来自正态分布的常用统计检验方法。它通过分析数据与正态分布的拟合程度来确定数据是否符合正态性。

Shapiro-Wilk 检验的基本原理

Shapiro-Wilk 检验的基本思想是将样本数据排序，并将排序后的数据与正态分布的理论分位数进行比较。检验统计量 WWW 的计算方式如下：

  1. 排序数据：首先将样本数据$ x1,x2,…,x_n$ 排序，得到排序后的数据y1≤y2≤⋯≤yny1≤y2≤⋯≤y_ny1≤y2≤⋯≤yn​ 。

  2. 计算检验统计量 WWW：

W=(∑i=1naiyi)2∑i=1n(yi−yˉ)2W = \frac{\left(\sum_{i=1}^{n} a_i y_i\right)^2}{\sum_{i=1}^{n} (y_i - \bar{y})^2}W=∑i=1n​(yi​−yˉ​)2(∑i=1n​ai​yi​)2​

  • 其中，$\bar{y} $是样本数据的均值。
  • 系数 aia_iai​ 是根据正态分布的理论分位数和样本大小 nnn 预先计算得到的常数。这些系数是通过对给定的样本大小进行数值优化得到的，使得 WWW统计量在样本来自正态分布时达到最大值。
  1. 检验结果的解释：
  • 检验统计量 WWW的取值范围在 0 到 1 之间，值越接近 1，表示数据越接近正态分布。
  • 与之对应的 ppp值用于判断数据是否符合正态分布。通常的假设检验方法是：
    • 原假设 H0H_0H0​：数据符合正态分布。
    • 备择假设H1H_1H1​ ：数据不符合正态分布。
  • 如果 ppp值较小（通常低于显著性水平，比如 0.05），则拒绝原假设，认为数据不符合正态分布。

Shapiro-Wilk 检验的优势

  • 适用于小样本：Shapiro-Wilk 检验被认为在样本量较小时仍然具有较高的检验效率。
  • 对分布尾部敏感：与其他正态性检验相比，Shapiro-Wilk 检验在检测偏离正态分布的尾部特征（如长尾、尖峰）时表现较好。

使用 Shapiro-Wilk 检验的注意事项

  • 样本大小：虽然 Shapiro-Wilk 检验适用于小样本，但对于特别大的样本（例如上千个数据点），它可能会过于敏感，检测出微小的偏离。此时，可能需要结合其他方法（如图形分析）进行综合判断。
  • 数据类型：Shapiro-Wilk 检验通常用于连续型数据。如果数据存在离散性或包含大量重复值，检验结果可能受到影响。

分类模型

逻辑回归

  1. 逻辑回归 (Logistic Regression)
  • 概述：逻辑回归是一种经典的二分类算法，适用于预测某个事件发生的概率。它假设输入变量与输出类别之间存在线性关系，并通过逻辑函数将线性回归模型的输出映射到概率值。
  • 应用：广泛用于二分类问题，如信用风险评估、病患是否患病等。
  1. 决策树 (Decision Tree)
  • 概述：决策树是一种树状结构的分类算法，通过对数据进行一系列的条件判断（节点分裂）来达到分类目的。每个节点根据某个特征将数据划分为不同的分支，直到叶子节点标记类别。
  • 应用：医疗诊断、市场细分、信用评分等。
  1. 支持向量机 (Support Vector Machine, SVM)
  • 概述：SVM 是一种监督学习模型，用于二分类问题。它通过寻找最优分割超平面将不同类别的数据点分开，并尝试最大化类间距离（最大化边界）。
  • 应用：图像分类、文本分类、生物信息学等。
  1. 随机森林 (Random Forest)
  • 概述：随机森林是一种集成学习算法，通过构建多个决策树来进行分类。它通过对每个决策树的预测结果进行投票，来确定最终的分类结果。随机森林可以减少过拟合并提高分类精度。
  • 应用：信用评分、疾病诊断、金融预测等。
  1. 梯度提升树 (Gradient Boosting Machines, GBM)
  • 概述：GBM 是一种逐步构建分类模型的算法。通过构建一系列弱学习器（如决策树），逐步减少模型预测误差。常见变种包括 XGBoost 和 LightGBM。
  • 应用：金融风险预测、网络安全、竞争性数据科学比赛等。

预测模型

  1. 线性回归 (Linear Regression)
  • 概述：线性回归是一种基础的回归分析方法，假设因变量与自变量之间存在线性关系。模型通过拟合一条直线来最小化观测值与预测值之间的误差。
  • 应用：房价预测、销售预测、市场趋势分析等。
  1. 岭回归 (Ridge Regression) 和 LASSO 回归 (Lasso Regression)

  • 概述：这两种回归方法是线性回归的扩展，加入了正则化项以防止过拟合。岭回归使用 L2L2L2正则化，LASSO 使用L1L1L1 正则化。

  • 岭回归：通过引入 L2L2L2正则化项，减少模型系数的绝对值，从而防止过拟合。

    LASSO 回归：通过引入 L1L1L1 正则化项，可以产生稀疏解，即部分系数被压缩为 0，从而实现特征选择。

  • 应用：特征选择、高维数据建模。

  1. 多元回归 (Multiple Regression)
  • 概述：多元回归是线性回归的扩展，用于研究多个自变量对一个因变量的影响。模型形式为：y=β0+β1x1+β2x2+⋯+βnxn+ϵy=β0+β1x1+β2x2+⋯+βnxn+ϵ=β0+β1x1+β2x2+⋯+βnxn+ϵy=β_0+β_1x_1+β_2x_2+⋯+β_nx_n+ϵy = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \dots + \beta_n x_n + \epsilon=β_0+β_1x_1+β_2x_2+⋯+β_nx_n+ϵy=β0​+β1​x1​+β2​x2​+⋯+βn​xn​+ϵy=β0​+β1​x1​+β2​x2​+⋯+βn​xn​+ϵ=β0​+β1​x1​+β2​x2​+⋯+βn​xn​+ϵ
  • 特点：
    • 时间依赖性：时间序列模型特别适合处理随时间变化的数据，捕捉趋势、周期性和季节性等特征。
    • 动态预测：能够对未来时刻的数据进行预测。
  • 应用：经济学、社会科学研究中常用来分析多个因素对结果的影响。
  1. 时间序列分析模型 (Time Series Analysis Models)

  • ARIMA (AutoRegressive Integrated Moving Average)：

    • 概述：ARIMA 模型是时间序列预测中常用的方法，适用于具有趋势和季节性特征的非平稳时间序列数据。
    • 应用：股票价格预测、经济指标预测、销量预测等。

  • SARIMA (Seasonal ARIMA)：

    • 概述：在 ARIMA 的基础上加入了季节性因素的建模，适用于具有显著季节性特征的时间序列。

  • Exponential Smoothing (指数平滑法)：

    • 概述：这类模型通过赋予时间序列数据中较近的观测值更大的权重来进行预测，常用方法包括简单指数平滑、霍尔特指数平滑和霍尔特-温特斯指数平滑。

    • 应用：短期需求预测、库存管理。

  1. 决策树回归 (Decision Tree Regression)
  • 概述：决策树回归通过将数据集分割成多个小区间来进行回归。它通过树状结构对数据进行分裂，适用于处理非线性关系。
  • 应用：市场预测、风险评估、价格预测等。
  1. 随机森林回归 (Random Forest Regression)
  • 概述：随机森林回归通过构建多个决策树来进行预测，利用多棵树的平均预测值来提高模型的稳定性和准确性。
  • 应用：多因子回归分析、金融预测、复杂非线性关系建模。
  1. 支持向量回归 (Support Vector Regression, SVR)
  • 概述：支持向量回归是一种扩展的支持向量机算法，通过构建一个能够容纳大部分数据点的“间隔”来进行回归分析。适用于高维数据和非线性回归问题。
  • 应用：股票市场预测、房地产估值、工程预测。

8.梯度提升回归 (Gradient Boosting Regression)

  • 概述：梯度提升回归是一种集成学习算法，它通过逐步添加弱学习器来改进预测精度。常用的实现包括 XGBoost、LightGBM 和 CatBoost。
  • 应用：大规模数据集的预测、比赛中的常用模型、复杂非线性问题。

9.主成分回归 (Principal Component Regression, PCR)

  • 概述：主成分回归结合了主成分分析 (PCA) 和线性回归，通过对特征进行降维后进行回归分析，适用于高维数据集。
  • 应用：高维数据预测、多变量统计分析。

现代优化算法

  1. 遗传算法 (Genetic Algorithm, GA)

特点：

  • 模拟自然选择：遗传算法模仿自然进化过程，通过选择、交叉、变异操作，逐步优化解。
  • 全局搜索能力强：适用于复杂的、多峰优化问题，能够在全局搜索空间内找到较优解。

实现过程：

  1. 初始化种群：

    • 随机生成一组解（染色体），每个解代表搜索空间中的一个点。

  2. 适应度评估：

    • 计算每个解的适应度值，衡量其在优化问题中的优劣。

  3. 选择操作：

    • 根据适应度值选择优良个体作为父代，常用方法有轮盘赌选择、锦标赛选择等。

  4. 交叉操作：

    • 将父代个体交叉组合生成新个体（子代），通过基因重组来探索解空间。

  5. 变异操作：

    • 对子代个体进行小概率的基因变异，增加种群多样性，防止陷入局部最优。

  6. 更新种群：

    • 用子代个体替换父代个体，形成新的种群，重复以上步骤直到满足停止条件。

  7. 输出最优解：

    • 经过多代进化，输出适应度最高的个体作为最优解。

  8. 粒子群优化 (Particle Swarm Optimization, PSO)

特点：

  • 群体智能：PSO 模仿鸟群觅食行为，个体（粒子）通过共享信息进行搜索。

  2. 计算适应度：

    • 计算每个粒子的位置对应的适应度值。

  3. 更新速度和位置：

    • 粒子根据其自身的最佳位置和全局最佳位置更新速度和位置：

    • 当达到最大迭代次数或满足收敛标准时，停止算法并输出最优解。

3. 模拟退火 (Simulated Annealing, SA)

特点：

  • 模拟物理退火过程：SA 算法模仿物理中金属退火的过程，通过逐步降低“温度”来寻找全局最优解。
  • 跳出局部最优：通过接受劣解的概率性跳跃，SA 能有效避免陷入局部最优。

实现过程：

  1. 初始化温度和解：
    • 设置初始温度 T0T_0T0​ 和初始解 x0x_0x0​。
  2. 迭代搜索：
    • 在当前解的邻域内随机生成一个新解 x’，计算其适应度变化 ΔE\Delta EΔE。
  3. 接受新解：
    • 如果 ΔE≤0\Delta E \leq 0ΔE≤0，接受新解 x′x'x′ 作为当前解；
    • 如果 ΔE>0\Delta E > 0ΔE>0，以概率 exp⁡(−ΔE/T)\exp(-\Delta E / T)exp(−ΔE/T) 接受新解，从而允许跳出局部最优。
  4. 降温：
    • 缓慢降低温度 T，常用的降温方式为 Tk+1=αTkT_{k+1} = \alpha T_kTk+1​=αTk​，其中 α\alphaα 是降温系数。
  5. 停止条件：
    • 当温度低于某个阈值或达到最大迭代次数时，停止算法并输出当前解作为最优解。

4. 蚁群算法 (Ant Colony Optimization, ACO)

特点：

  • 模拟蚂蚁觅食行为：ACO 模仿蚂蚁通过信息素（Pheromone）引导的觅食行为，适用于组合优化问题。
  • 并行性强：算法可以并行执行，适合大规模问题。

实现过程：

  1. 初始化信息素：
    • 初始化所有路径上的信息素强度为常量。
  2. 构建解：
    • 每只蚂蚁根据信息素和启发式信息选择路径，逐步构建一个解。路径选择的概率与信息素强度和路径吸引力相关。
  3. 更新信息素：
    • 蚂蚁完成解构建后，根据路径质量更新信息素： τij(t+1)=(1−ρ)τij(t)+∑k=1mΔτijk\tau_{ij}(t+1) = (1 - \rho) \tau_{ij}(t) + \sum_{k=1}^{m} \Delta \tau_{ij}^kτij​(t+1)=(1−ρ)τij​(t)+∑k=1m​Δτijk​ 其中 ρ\rhoρ 是信息素挥发系数，Δτijk\Delta \tau_{ij}^kΔτijk​ 是第 k 只蚂蚁在路径 (i,j)(i, j)(i,j) 上的增量。
  4. 迭代搜索：
    • 重复构建解和更新信息素的过程，逐步强化优良路径的信息素浓度。
  5. 停止条件：
    • 达到最大迭代次数或信息素强度收敛时，停止算法，并输出最优解。
